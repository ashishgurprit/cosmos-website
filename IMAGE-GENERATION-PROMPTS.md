### Hero Images (1200x630px)
- Blog title overlaid on image
- Cosmos Web Tech logo in top left corner
- Author name "Ashish Ganda" in bottom right
- Professional, modern design
- Brand colors: Navy (#1e3a5f), Blue (#3b82f6), White

### Content Images (800x450px)
- Support the main topic of the blog
- Professional business/tech aesthetic
- Western Sydney local flavor when relevant

### Infographics (800x1200px)
- Vertical format, mobile-friendly
- Key statistics and takeaways
- Step-by-step guides or checklists
- Brand colors throughout

## Implementation Notes

### Tools for Creating Images:
1. **Canva Pro** - Best for hero images with text overlays
2. **Figma** - Professional mockups and infographics
3. **Adobe Express** - Quick professional graphics
4. **Photopea** - Free Photoshop alternative
5. **Remove.bg** - Clean up image backgrounds

### Hero Image Template Requirements:
- Dimensions: 1200x630px
- Cosmos Web Tech logo: Place at 50px from top, 50px from left
- Title: Centered or left-aligned, 48-64pt bold font
- Author: "By Ashish Ganda" at bottom right, 18-24pt
- Brand colors: Navy #1e3a5f, Blue #3b82f6

### Brand Assets:
- Logo: Network constellation design (already in site)
- Fonts: Inter (body), Plus Jakarta Sans (headings)
- Colors: Navy Dark (#1e3a5f), Blue (#3b82f6), Grey (#64748b), White (#ffffff)

### File Naming Convention:
```
[slug]-hero.jpg (1200x630)
[slug]-content.jpg (800x450)
[slug]-infographic.jpg (800x1200)
```

Example:
